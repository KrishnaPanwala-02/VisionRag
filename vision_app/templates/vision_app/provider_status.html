How the load balancer works

VisionRAG distributes requests round-robin across all non-rate-limited cloud providers. If a provider returns HTTP 429, it's cooled off for 60 seconds before retrying. Ollama is always last in the chain — it's the unlimited local fallback and never rate-limited.
